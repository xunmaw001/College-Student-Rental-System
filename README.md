# College-Student-Rental-System
基于 Spring Boot + Vue 的大学生租房系统，是专门服务于大学生的租房平台。它支持房源信息展示、筛选、搜索，以及预订、租赁管理等功能。前端 Vue 提供简洁易用的操作界面，方便大学生操作；后端 Spring Boot 确保系统稳定，保障房源和用户信息的安全存储和高效处理，为大学生提供便捷、安全的租房体验，解决其租房需求。
